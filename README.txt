# Cold War Influence: U.S. vs. U.S.S.R.

A GitHub Pages site comparing how the United States and Soviet Union maintained influence during the Cold War, with focus on proxy wars like the Korean War, Angolan Civil War, and Nicaraguan Revolution.

## Features

- Interactive timeline comparing strategies
- Focus on three major proxy wars
- Educational quiz and interactive elements
- Properly cited sources

## Technical Details

- Built with HTML, CSS, and JavaScript
- Uses Bootstrap 5 for responsive design
- All images properly attributed and hosted locally

## Sources

All sources are properly cited in the website's bibliography section.
